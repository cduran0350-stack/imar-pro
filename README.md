# İmarPRO Professional

Profesyonel şehir plancıları, mimarlar ve gayrimenkul geliştiricileri için tasarlanmış bir imar hesaplama ve analiz yazılımıdır. 

## Özellikler
- TAKS ve KAKS (Emsal) bazlı net/brüt alan hesaplamaları
- Otomatik profesyonel mimari değerlendirme raporu
- Arayüzden bağımsız temiz yazdırma modu
- Tema: Zümrüt Yeşili ve Koyu Lacivert (Modern Arayüz)

## Teknolojiler
- HTML5, CSS3, ES6+ JavaScript (Bağımlılık içermez)
